The Mission

Our project aims to facilitate medical examinations and reduce the wasted time, effort, and transportation money. We designed a system composed of wearable sensors used to measure temperature, Blood pressure, ECG and pulse rate. The data resulting from the sensors will be sent to a server in the hospitals via Raspberry Pi. the server will use an SVM algorithm, which is used to classify all the data and compare it to reach the desired and proper arrangement of data. Our prototype was tested to be insured that it works efficiently and achieves the desired results. We found a relative sensor error was about 2%, which can be neglected. The prototype was eco-friendly as it did not impact the environment or humans negatively.
